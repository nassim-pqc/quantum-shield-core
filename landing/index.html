ML-KEM-768

NIST-standard key encapsulation. 1184 bytes public key, 2400 bytes secret key. Level 3 security.

AES-256-GCM

Authenticated encryption with AAD binding. Context-linked ciphertext for zero-trust workloads.

Rust Core

Memory-safe crypto engine. PyO3 bindings. Panic-abort on release. Constant-time HMAC.

Signed Audit

Append-only, HMAC-SHA256 signed audit trail. Key rotation. Hash-chain ready.

Use Cases

🔐 Data-in-Transit Protection

Tamper-evident, HMAC-signed audit logs for NIS2, PCI-DSS, SOC 2, and ISO 27001 compliance.

Stateless design — no private keys stored server-side. Context-bound AAD for workload identity.
