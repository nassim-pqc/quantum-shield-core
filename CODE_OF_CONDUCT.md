# Code of Conduct — Quantum Shield Core

## Our pledge

We are committed to a professional, harassment-free environment for everyone participating in this project — contributors, evaluators, acquirers, and integrators.

## Expected behavior

- Use welcoming and inclusive language
- Respect differing viewpoints and experiences
- Accept constructive criticism gracefully
- Focus on what is best for the project and its users

## Unacceptable behavior

- Harassment, trolling, or discriminatory comments
- Publishing others' private information without permission
- Other conduct inappropriate in a professional setting

## Enforcement

Project maintainers may remove, edit, or reject contributions and communications that violate this code of conduct.

Report issues to the contact listed in [SECURITY.md](SECURITY.md).
