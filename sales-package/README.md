## What Is This Package?

This sales package contains everything needed to evaluate **Quantum Shield Core** — an enterprise post-quantum cryptographic microservice — without searching through the full repository.

**In 5 lines:**
- Enterprise-grade post-quantum encryption using NIST-standard ML-KEM-768 (Kyber768) + AES-256-GCM
- Stateless architecture: no user private keys stored server-side
- Pluggable KMS: AWS KMS, Azure Key Vault, HashiCorp Vault
- Two SDKs: Python and Go
- Full observability: Prometheus, OpenTelemetry, structured JSON logs

**GitHub**: https://github.com/nassim-pqc/quantum-shield-core
